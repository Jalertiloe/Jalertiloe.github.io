Paddle Ball This is my first project I completed with Girls Who Code in 2016. During our first week we used a free programming language, created by MIT called Scratch. This is a simple paddle game in which the score and speed increases every time the unicorn touches the paddle.
Pig This is the second project that I completed with Girls who code in 2016. This was also created during our first week in Scratch. The point of this game is for player one to get predetermined number of points before player two while avoiding rolling a one with negates all points earned in the current round. y
Jukebox This was also created during the first week of Girls Who Code. Music was added note by note in order to create songs which were stored as lists. By pressing the numbered buttons you add the list to a larger list. After added or deleting songs, you execute the play command.
Obamicon This was created during the second week of Girls Who Code as an introduction to Python and PILLOW. The goal was to emulate the "Obama Hope" poster.
Snow Animation This was created during the second week of Girls Who Code as an introduction to Pygame. The goal was to create a procedure that randomly created circles in such a way that they appeared to be moving downward.
Bouncing Ball This was created during the second week of Girls Who Code. This encorparated creating two circles with randomly genrated colors for the objects and the background while allowing them to "bounce" off of the borders .
