Paddle Ball Scratch
This is my first project I completed with Girls Who Code in 2016. During our first week we used a free programming language, created by MIT called Scratch. This is a simple paddle game in which the score and speed increases every time the unicorn touches the paddle.
Pig Scratch
This is the second project that I completed with Girls who code in 2016. This was also created during our first week in Scratch. The point of this game is for player one to get predetermined number of points before player two while avoiding rolling a one with negates all points earned in the current round.
Jukebox Scratch
This was also created during the first week of Girls Who Code. Music was added note by note in order to create songs which were stored as lists. By pressing the numbered buttons you add the list to a larger list. After added or deleting songs, you execute the play command.
Obamicon Python

Snow Animation Python
This was created during the second week of Girls Who Code as an introduction to Pygame. The goal was to create a procedure that randomly created circles in such a way that they appeared to be moving downward.
Bouncing Ball Python
This was created during the third week of Girls Who Code. This encorparated creating two circles with randomly genrated colors for the objects and the background while allowing them to "bounce" off of the borders .
City Scroller Python
Also created during the third week of Girls Who code, was the begining of a game created in Python with the pygame library. This program randomly generated rectangles that moved across the screen without changing size to simulate an abtract cityscape.
Runner Game Python
On the final day of week three, we combined the skills learned in the bouncing ball activity and the city scroller activity to create a functional "runner game". The game initially used the mouse as the main controls and later the arrow keys were used as controls. During this project, we learned how to implement sprites and to use classes from outside of the file.
Robot Dance C++
After only two days of learning C++ in Arduino, ten groups of four girls programmed robots to dance and flash LEDs to chosen music. The one depicted to the left was the one created by my group, Groffsauce, created a robot in the likeness of King George III from Hamilton An American Musical to the song You'll be Back . This robot was featured in a segment of People to People.
Robot Whiskers C++
On the final day of week four we devled into the basics of machine learning. This project displayed a more practical use of movement in a robot. We used whisker sensors to guide the robot through a maze.

I wanted to make something that would have an Impact on people so, I created a game in which you destroy negative comments siad about people daily.
